# 《尚小噜 ❤️ 王小噜》

这是一个使用 HTML + CSS + JavaScript 制作的电影级恋爱纪念网站，适合直接部署到 GitHub Pages。

## 功能特点
- 星空背景
- 爱心粒子
- 樱花飘落
- 页面滚动动画
- 自动计算恋爱天数
- 照片时间轴与照片墙
- 点击照片放大
- 情书内容
- 互动按钮与烟花、爱心和彩带动画
- 兼容手机与桌面端

## 部署方式
1. 将项目上传到 GitHub 仓库。
2. 打开仓库设置中的 Pages。
3. 选择默认分支并保存。
4. 访问生成的 GitHub Pages 地址即可。

## 说明
- 需要将照片放入 images 文件夹中，文件名建议为 1.jpg、2.jpg、3.jpg 等。
- 当前页面会自动读取 images 目录下的图片进行展示。
